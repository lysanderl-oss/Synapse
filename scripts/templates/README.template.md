## Quick Start (5 minutes)

### 1. Clone the repository

```bash
git clone https://github.com/lysanderl-glitch/synapse.git
cd synapse
```

### 2. Personalize (3 replacements in CLAUDE.md)

Open `CLAUDE.md` and use `Ctrl+H` to replace:

| Find | Replace with | What it does |
|------|-------------|--------------|
| `Lysander` | Your AI CEO's name | How the CEO introduces itself |
| `{{PRESIDENT_NAME}}` | Your name | How the AI addresses you |
| `{{COMPANY_NAME}}` | Your company/team name | Organizational identity |

### 3. Open in Claude Code

Open the `synapse` folder in Claude Code. Send any message — your AI team is ready.

## How It Works

Synapse implements 5 Harness Engineering principles:

1. **Guides + Sensors** — Feedforward control (what to do) + feedback control (check if done)
2. **Unchained execution pipeline** — Every task passes fixed checkpoints, no skipping
3. **Role separation** — CEO manages, experts execute. Never crossed.
4. **4-level decision system** — 95% of decisions resolved below human escalation
5. **Auditable agent capabilities** — Every agent scored ≥90/100 or flagged for upgrade
