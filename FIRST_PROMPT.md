# 首次使用引导词

> 打开 Claude Code 并加载 Synapse 文件夹后，复制以下内容发送。
> （已完成 CLAUDE.md 个性化配置后使用）

---

## 引导词（直接复制粘贴）

```
你好，请你按照 CLAUDE.md 中的执行链规范，以 AI CEO 身份问候我，
并告诉我你当前加载的团队成员有哪些。
我是第一次使用这套系统，请简要介绍如何发起一个任务。
```

---

## 日常使用模板

### 启动新任务
```
[CEO名] 我今天的目标是 [描述目标]，请按执行链推进。
```

### 召集智囊团分析
```
[CEO名] 请召集智囊团，帮我分析 [问题/决策]
```

### 执行具体任务
```
[CEO名] 请 [团队名称] 帮我完成 [具体任务]
```

### 知识沉淀
```
[CEO名] 请 OBS 团队将本次 [项目/任务] 的经验沉淀到知识库
```

### 内容/报告生产
```
[CEO名] 请内容团队帮我撰写 [报告/提案/文档]
```

### 升级体系
```
升级 Synapse
```
