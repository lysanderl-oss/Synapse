## 需求背景

自动抓取关注的微信公众号新文章，推送到知识库或阅读队列，减少手动打开微信查看的频率。

## 方案推荐

**推荐架构**：We-MP-RSS + n8n + Claude

| 组件 | 作用 |
|------|------|
| We-MP-RSS | 将微信公众号转化为 RSS Feed |
| n8n | 定时拉取 RSS，触发后续工作流 |
| Claude | 文章摘要生成、分类标注 |
| Obsidian/Notion | 知识沉淀目标 |

## 备选方案

- **RSSHub**：开源 RSS 生成器，支持微信公众号，需自托管
- **WeRSS**：商业服务，稳定性较好，有费用
- **直接爬虫**：需维护登录状态，风险较高
