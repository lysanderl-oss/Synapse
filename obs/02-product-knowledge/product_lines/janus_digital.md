# Janus Digital — Agent 产品与服务产品线

## 产品线定义

Janus Digital 面向垂直行业客户，以 **Agent 产品与服务** 为核心交付物，
依托 Synapse 内核（Multi-Agent 组织 / 执行链 / CEO Guard）提供
高可信、可交付、可定价的 Agent 解决方案。

区别于通用 AI 平台，Janus Digital **不做模型、不做基础设施**，
专注"应用 + 服务 + 垂直场景"三位一体的 Agent 落地能力。

## 市场依据

- **Stanford AI Index 2026**：Agent 任务成功率从 12% 跃升至 66%，质变拐点已到
- **WebArena 基准**：开放式 Web Agent 在真实任务上已具备可交付能力
- **Q1 2026 AI 风投**：AI 占风投总量 **80%**，垂直行业 AI SaaS 同比 **+28%**，跑赢横向平台 3 倍
- **哑铃效应**：500M+ 巨轮 vs 3M- 微轮，中间层真空，Janus Digital 需清晰定位
- 综合结论：Q2 2026 是 Agent 商业化的关键窗口，错过将面临巨头挤压

## 差异化定位（利用 Synapse 内核资产）

| Synapse 内核资产 | Janus Digital 外化价值 |
|------------------|----------------------|
| Multi-Agent 44 人组织 | 多角色协同可直接打包为"虚拟团队"服务 |
| 执行链四级决策 | Agent 交付质量可审计，企业客户可信 |
| CEO Guard | Agent 行为边界硬约束，降低失控风险 |
| OBS 知识库 | 领域知识沉淀机制，支持垂直行业快速 Onboarding |

**差异化优势**：不是"卖 Agent API"，而是"交付一个受治理的虚拟团队"。

## 与 Enterprise Governance 产品线的关系

- **Janus Digital**：做 Agent（产品 + 服务）
- **Enterprise Governance**：管 Agent（治理方案 + 框架）
- 两条产品线**互补而非竞争**：
  - 客户买 Janus Digital → 自带治理（Enterprise Governance 内嵌）
  - 客户买 Enterprise Governance → 可治理自建 Agent 或第三方 Agent
- 联合销售场景：大客户同时采购 Agent 产品 + 治理方案
