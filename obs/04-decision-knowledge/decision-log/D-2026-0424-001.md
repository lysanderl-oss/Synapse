# D-2026-0424-001：SSOT-I18N 战略（双语化 + 公开 + 学院徽章）

## Decision

总裁批准启动 SSOT + i18n 五阶段战略：synapse-core 立即公开（决策①A）、学院首期仅做完成徽章（决策②A）、首期不出镜由 Multi-Agents 全程交付（决策③B）。

## Context

阶段 1 SSOT（单一真相源）工作被"synapse-core 是否公开"卡住。同时 lysander.bond 网站缺英文版，海外受众无法触达。学院课程在产品维度需选择"完成徽章"vs"正式认证"两条路。三议题相互绑定，必须同步裁决。

## Options Considered

- **决策①** synapse-core 公开时机：A 立即 push / B 脱敏后 / C 延迟
- **决策②** 学院认证体系：A 仅徽章 / B 正式认证 / C 不做
- **决策③** 总裁个人 IP 出镜：A 出镜 1 门 / B 不出镜 / C 延后

## Selected

- **①A** 立即 push 公开（Lysander 建议 A，总裁同意）
- **②A** 首期仅做完成徽章（Lysander 建议 A，总裁同意）
- **③B** 不出镜，纯 Multi-Agents 交付（Lysander 建议 A，被否决）

**理由**：总裁认为"不依赖个人 IP 的可复制性"高于"创始人品牌势能"。这是 Lysander 学到的关键纠偏点。

## Consequences

- Positive：解除阶段 1 所有阻塞；启动双语全量化；建立"无创始人依赖"的体系样本
- Negative：失去早期"创始人故事"营销弹药
- Reversibility：①不可逆（已公开）/ ② 易（升级正式认证只需扩展机制）/ ③ 中（后期可补出镜）

## Follow-up Actions

- 阶段 1-5 由 Lysander + 智囊团闭环
- 每阶段完成简报总裁
- 未来认证商业化如启动，独立 L4 上报
