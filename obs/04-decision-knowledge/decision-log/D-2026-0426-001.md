# D-2026-0426-001：战略改造审批（产品 / 推广 / 授权三方案整合）

## Decision

总裁批准网站内容整体性改造 + 推广策略调整 + 复用授权策略的三方案整合执行：锁定"开源体系 + 咨询服务"主线，内部 Dogfooding 优先，B+D 授权分阶段落地，整合 16 项 P0 立即执行。

## Context

三大议题（产品 / 市场 / 治理）同时暴露：
- 网站叙事混乱（Synapse / Forge / Academy 三套品牌叠加）
- 推广早于产品成熟度
- 复用没有法律护栏

智囊团评审（strategy_advisor + execution_auditor + decision_advisor）三方案高度一致、无方向性冲突，9 项 L3 已由 Lysander 自决，7 个 L4 决策点上呈总裁。

## Options Considered

7 个 L4 决策点（含品牌统一、BSL 选型、Advocate 机制、推广扳机阈值等），每点 A/B/C 选项详见审批包正文。

## Selected

主线决策：**"开源体系 + 咨询服务"** 单一故事 + Dogfooding 优先 + 法律护栏先行 + 删减优先于新增。

具体 7 项 L4 裁决归档于 `2026-04-26-strategic-overhaul-approval-package.md`，**关键裁决**：
- 品牌统一为 Synapse（去 Forge 包装层）
- BSL-1.1 选型 + 网站文案精确表述为 "source-available, open by 2030"
- Advocate 4 级机制（L1 Active → L4 Co-creator）
- 对外重启硬扳机：内部 NPS≥50 + DAU/WAU≥0.5 + 可公开案例≥3 + L3+ Advocate≥5（同时满足）

## Consequences

- Positive：消除叙事冲突；建立法律护栏；可观测可回滚的 Dogfooding 闭环
- Negative：短期推广增长暂停；需要承受"内部反馈闭环"窗口期
- Reversibility：中（推广可重启，扳机条件可调；BSL 选型 4 年内可调整）

## Follow-up Actions

- 治理批次（4 项）1 工作日完成
- 网站改造批次（6 项）2-3 工作单元
- 推广暂停批次（3 项）0.5 工作日
- 测量基线批次（3 项）网站改造后启动
- 各批次串行依赖：治理 → 网站 → 推广 → 测量
